Raspberry Pi 3 B+ WoA

This is a sequence of steps to create an Erlang development environment inside Windows Subsystem for Linux on Rapsberry Pi 3 B+. I'm using Windows 10 Insider build 18895 for both AMD64 and ARM64 host and device platforms.

TOC

— WoA: Install Windows on SD card
— WIN: Setup Windows
— PERF: Tune up Performance
— WSL: Install Linux
— ERL: Install Erlang

WoA

First of all I want to say even if you bought the device with radiators, SD card, and power adapter, this is still if not enough in case you don't have SD card reader. The general idea is that WoA Deployer takes original WIM image and by using DISM commands, first apply this image to SD card, then apply drivers over vanialla Windows 10 ARM64.

— WOA-Project/WOA-Deployer-Lumia — 2.5.4

WIN

After flashing USB perform regular Windows Setup procedure and set up to boot from USB always at first in EFI.

PERF

As for Windows this is personal preferences, but in reality RP3B+ is better to use cmd.exe as a Shell instead of explorer.exe. Also diable any GPU GUI effects, disable Windows Defender in gpedit.msc, disable Windows Update, Windows Search, everything you find allocated CPU or MEM in taskmgr.exe. After all preparation you should see at most 10% CPU consumption in idle as GPU drivers is not accessible in RP3B+ WoA yet.
